# 上海吳語拼音輸入方案


配方：℞ **NGLI/rime-wugniu_zaonhe**

基於 [Rime](https://rime.im/) 的上海吳語拼音輸入方案。

含上海（中派）、上海（老派）和松江三種口音的輸入方案。

拼音方案爲 [吳語學堂式拼音](https://github.com/NGLI/rime-wugniu_zaonhe/wiki/音系及拼音方案)，同時兼容 [吳語協會式拼音](http://wu-chinese.com/romanization/)。

參考了以下項目：
